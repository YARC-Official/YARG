Native BASS binaries must be placed in their correct folders for Unity to load them correctly.
Not installing BASS will cause the game to fail to load/play audio

Version to download is 2.4.17 (Released 12th October 2022)
Download: https://www.un4seen.com/

Libraries required:
BASS
BASSOPUS (addon)
BASSmix (addon)
BASS FX (3rd party addon)
